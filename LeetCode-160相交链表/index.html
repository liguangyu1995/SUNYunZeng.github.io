LeetCode 160.相交链表

2019-11-04 Algorithm Other Algorithm 0 评论 字数统计: 408(字) 阅读时长: 2(分)

题目

编写一个程序，找到两个单链表相交的起始节点。

思路 + 代码

1. 最容易想到的，两层遍历求解。

时间复杂度O(M*N)
空间复杂度O(1)

2. 利用Map记录一个链表的每个节点，第二个链表寻找第一次出现的节点。

时间复杂度O(M+N)
空间复杂度O(M)或O(N)

3. 或者两个指针，分别从两个链表的头结点开始，当一个节点遍历到尾部时换到另一个链表头部。

即利用 a+all+b = b+all+a，也就是两个指针走的路程一样。

时间复杂度O(M+N)
空间复杂度O(1)
