枚举类替代整形常量

枚举类的基本形式如下：

enum关键字指明一个类继承 abstractEnum。

该类有两个重要的方法：

  1. ordinal(). 返回整形序列值（从0开始）

  2. Enum(). 只能被继承Enum的类调用的构造方法，枚举类中每个实例调用该方法实现赋值，每个实例的构造方法都是私有的。

枚举类的特征
1. 不能被继承其它类也不能被其他类继承，但可以实现接口，对枚举类内部实例进行组织。

2. 类中每个实例可以重写其中定义的抽象方法

3. 可利用枚举实现单例

4. 调用values()方法遍历枚举类型中的每个实例
